header - Cabecera de un sitio web o de una seccion
main - Define la seccion principal del documento, solo puede existir una etiqueta main por documento
footer - Pie de pagina de un sitio web o de una seccion
nav - Representa una navegacion o botonera
article - Representa una seccion autocontenido (que por si sola se explica sin ver lo demas de la pagina)
aside - Representa contenido complementario o secundario (o publicidad)
section - Rrepresenta una secci칩n de contenido gen칠rico
